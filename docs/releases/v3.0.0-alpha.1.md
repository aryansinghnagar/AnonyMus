## 1. Overview & Architecture Modernization

AnonyMus `v3.0.0-alpha.1` marks the official transition to a modern **FastAPI v3 backend architecture** backed by **SQLAlchemy asynchronous engines (SQLite WAL mode)**, **Alembic schema versioning**, and a **SolidJS + Vite + TypeScript** web client PWA bundle.

This release introduces post-quantum hybrid double-ratcheting, sealed-sender envelope privacy, MLS RFC 9420 group keying infrastructure, turn-key node launchers, and bit-for-bit reproducible build attestation manifests.

---

## 2. Key Release Highlights

* **Turnkey Client Launcher (`anonymus-launcher.py`)**: One-command client boot that checks local Tor SOCKS5 proxy status (`127.0.0.1:9050`), applies database migrations, starts the FastAPI server on port 5001, and opens the local web client interface.
* **Post-Quantum Hybrid Double Ratchet**: Hybrid derivation combining X25519 DH key exchange with NIST FIPS 203 ML-KEM-768 (Kyber768) post-quantum encapsulation via HKDF-SHA256.
* **Sealed-Sender Envelope Privacy**: ECIES outer envelope encryption protecting sender identities from pairwise relays.
* **Uniform Payload Padding**: Pad-to-2KB uniform block sizing with random byte jitter to mitigate traffic-size side channels.
* **MLS RFC 9420 Scalable Group Engine**: High-level TreeKEM group state engine (`MLSGroupContext`, `MLSKeyPackage`) supporting $O(\log N)$ group rekeying.
* **Zeroization Panic Wipe (`obliviate`)**: Instant local and remote database file zeroization overwriting state with cryptographic noise (`os.urandom`).
* **Reproducible Build Verification (`scripts/verify_reproducible_build.py`)**: Generates Cosign-compatible `reproducible_attestation.json` manifests detailing SHA-256 digests across build artifacts.

## 3. Quick Start Guide

### Running Local Client Node
```bash
# Clone repository
git clone https://github.com/aryansinghnagar/AnonyMus.git
cd AnonyMus

# Create virtual environment and install dependencies
python -m venv venv
venv/Scripts/pip install -r requirements.txt

# Boot AnonyMus client node
python anonymus-launcher.py
```

### Self-Hosting Relay Stack
```bash
# Start pairwise relay with Caddy auto-TLS and Redis ephemeral queue
docker compose up -d
```
