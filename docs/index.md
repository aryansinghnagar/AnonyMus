# AnonyMus Documentation Hub

Welcome to the official documentation site for **AnonyMus**, a metadata-resistant, privacy-preserving instant messaging platform.

AnonyMus is engineered to resist passive and active network analysis by operating either over a federated/relay structure or natively inside a Peer-to-Peer network over Tor Hidden Services.
